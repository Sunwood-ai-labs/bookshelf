## 📖 概要 (About)

**Bookshelf** は、ローカルのマンガや書籍データを「置くだけ」で爆盛れ本棚に変身させるWebアプリだよ〜！💖

「整理とかマジめんどい」って？ それな！
だから **Folder-as-Book** 構造を採用。フォルダ名がそのままタイトルに、中の画像がそのままページになるから、管理コストは実質ゼロ！

さらに **Manga Stack UI** で、積読（つんどく）すらもインテリアにしちゃうよ📚✨

---

## 💖 推しポイント (Features)

### 1. Manga Stack UI 📚
ただ並べるだけじゃつまんないじゃん？
本を積み上げたような「Manga Stack」デザインで、視覚的にもテンション上がるUIに仕上げたよ！
ダークモード対応で、夜更かし読書も目に優しい🌙

### 2. Folder as Book 📂➡️📖
面倒なDB登録とかインポート作業とか、一切ナシ！
所定のフォルダにマンガのフォルダをポンって置くだけ。
あとはアプリが勝手に認識して、いい感じに並べてくれるよ〜！神！

### 3. Immersive Reader 👓
読むときは余計なもの一切ナシ。
没入感ヤバめのリーダーで、ストーリーの世界にどっぷり浸かっちゃお🫶

## 🚀 始め方 (Getting Started)

とりあえず動かしてみよ？ 3ステップで完了！

### 1. クローンしてくる
```bash
git clone <repo-url>
cd bookshelf
```

### 2. 依存関係を入れる
```bash
cd app
npm install
```

### 3. 起動！
```bash
npm run dev
```

ブラウザで `http://localhost:5173` を開けば、そこはもうあなただけの図書館🏰✨
